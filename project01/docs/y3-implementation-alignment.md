## 必须保持的核心口径

- 我方左侧：`741 / 852 / 963`。
- 敌方右侧：`147 / 258 / 369`。
- 前排 = `1/2/3`，中排 = `4/5/6`，后排 = `7/8/9`，侧翼 = `1/4/7` 或 `3/6/9`。
- 每名英雄带小兵队伍，队长等于英雄。
- 玩家队长死亡后小兵溃败，NPC 队长死亡后小兵继续战斗。
- 每名英雄 6 技能，3 主动 + 3 被动，通过升星解锁。
- 英雄进入战斗自动携带全部已解锁技能。
- 已解锁主动技能始终自动释放，不提供按英雄关闭自动释放的开关。
- 备战阶段展示 3 个军令报价，玩家可用局内军资购买多张军令卡并付费刷新。
- 主动军令在战斗 HUD 中由玩家释放；被动军略在本局剩余波次持续生效，并在军略图鉴中查看。
- 卡牌、军资、Set 解锁和合成结果均为局内状态，本局结束后清空；英雄主动技能仍只自动释放，不进入军令卡池。
- 具体卡牌目录、经济数值、主动槽位和使用限制尚未定案，不得继续按 V0 固定三卡规则实装。
- 旧版 6 选 4、旧装备、旧战备、旧羁绊不得作为首轮原型通过条件。

## 首轮原型优先验证

1. 阵容与 3x3 站位。
2. 备战军令市场与战斗 HUD 的 V1 交互框架。
3. 备战状态。
4. 通关奖励。
5. 失败结算。
6. 首批 10 英雄专属循环与军令卡系统是否可读、可测、可落表。
