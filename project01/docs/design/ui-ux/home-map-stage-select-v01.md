## 废弃原因

v01 曾用于探索主界面地图选关的信息结构，但后续视觉方向已经调整为更接近《三国志14》的现代历史策略 UI。v01 不再作为后续出图参考，避免重新带回旧版中偏厚重、偏陈旧的界面倾向。

## 保留价值

仅保留以下玩法与结构结论：

- 关卡切换采用同层级地图节点。
- 已解锁关卡可以回打，不被新关卡覆盖。
- 出战阵容规则为默认 2 人、最多 4 人。
- 主界面阵容区只做摘要，不放重复的“调整”按钮。

以上结论已经迁移并整理到 [home-map-stage-select-v02.md](home-map-stage-select-v02.md) 和 [ui-guidelines.md](ui-guidelines.md)。后续设计请读取 v02 文档。
