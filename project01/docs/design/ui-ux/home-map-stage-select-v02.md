# 主界面 / 地图选关入口 v02

## 资源

- 预览图：`assets/ui/preview/home-map-stage-select-v02.png`
- 状态：已确认
- 界面：主界面 / 地图选关入口
- 设计方向：现代历史策略 UI，参考《三国志14》的清爽战略地图感

## 界面定位

主界面不是传统功能大厅，也不是运营活动页。它承载玩家进入一局挑战前的核心判断：

- 当前选择哪一个关卡。
- 已解锁关卡是否可以回打。
- 当前关卡的进度、难度、推荐应对方向和奖励预期。
- 当前出战队伍是否满足挑战准备。
- 下一步是继续挑战、调整阵容，还是进入局外成长。

## 已确认设计规则

### 同层级地图选关

关卡切换采用三国大陆地图上的节点表达。所有已解锁关卡处在同一层级，玩家可以点击地图节点切换关卡信息。

这种设计用于避免误解为“打完关卡 1 后关卡 2 覆盖关卡 1”。已解锁关卡应保留可回打状态。

### 右侧关卡信息面板

选中关卡后，右侧展示该关卡的主要信息：

- 关卡名称。
- 当前进度或历史最好进度。
- 难度状态。
- 推荐应对关键词。
- 主要奖励或解锁提示。
- 最高优先级主操作。

主操作以“继续挑战”或等价挑战按钮为最高权重。同一界面不应再出现第二个同权重的挑战按钮。

### 阵容摘要

主界面只展示阵容摘要，不在摘要内放重复的“调整”小按钮。调整阵容入口应位于系统入口区，避免一个功能在同一屏里出现两个触发点。

出战人数规则固定为默认 2 人，最多 4 人。主界面摘要可以显示 4 个槽位，但不要把它做成完整阵容编辑器。

### 局外入口克制

主界面入口聚焦：

- 调整阵容。
- 武将养成。
- 兵书成长。
- 必要的设置或资源查看。

不在 MVP 主界面加入商城、礼包、活动、每日任务、充值、图鉴一级入口。

## 视觉规范

- 背景使用战略地图、山川城池、行军路线等可读元素。
- 面板使用浅色石瓷/宣纸感，但不要变成厚重卷轴。
- 重点色使用克制的朱红、金铜或墨色，不使用黑金页游按钮。
- 地图节点必须看起来可点击，而不是纯装饰。
- 画面整体应明亮、清爽、有策略游戏气质。

## 后续出图注意

- 若做阵容调整界面，需要继承“默认 2 人 / 最多 4 人”的规则。
- 若做关卡详情或难度选择界面，需要继承“已解锁关卡可回打”的规则。
- 若做局外成长界面，需要保持入口克制，不扩展成运营大厅。
