# 兵书成长 v02

## 资源

- 预览图：`assets/ui/preview/meta-upgrade-v02.png`
- 状态：已确认
- 界面：兵书成长
- 入口：局外主界面 / 地图选关页的“兵书成长”

## 界面定位

兵书成长是局外弱成长界面，用于提供长期目标和构筑稳定性。它不直接出售强战力，不阻断默认 2 人位的基础通关可能。

玩家在这里完成：

- 查看四条兵书成长路线。
- 选择一个节点查看等级、效果、消耗和解锁条件。
- 消耗资源升级一个弱成长节点。

## 已确认设计规则

### 一页四条路线

本版不使用切页，而是在一张页面中展示四条成长路线：

- 通用。
- 战备。
- 奖励。
- 阵营。

这样玩家可以一眼看到兵书系统的完整结构，减少在局外成长页中来回切换。

### 不是强前置技能树

节点之间可以用淡墨细线表达“同类成长方向”，但不使用箭头、阶梯式强依赖或必须前置满级的视觉语言。

当前兵书节点的解锁来自：

- 初始开放。
- 关卡或难度进度。
- 阵营使用次数。
- 资源消耗。

不表达为“前一个节点升满后才能解锁后一个节点”。

### 顶部资源栏保持克制

全局顶部资源栏只展示：

- 铜钱。
- 兵书残页。
- 设置入口。

`阵营札记` 和 `试炼印记` 不放入全局顶部栏，只在兵书页内部作为特殊资源说明或节点消耗出现。

### 右侧节点详情

右侧详情面板展示当前选中节点：

- 名称。
- 类别。
- 当前等级 / 最大等级。
- 当前效果。
- 下一级效果。
- 消耗资源。
- 解锁条件。
- 弱成长边界提示。

### 特殊资源说明

页面内可用轻量说明解释：

- 阵营札记：用于阵营路线。
- 试炼印记：用于高阶节点。

这两个资源不应被包装成常驻主货币，也不应引导商城或充值联想。

## 视觉规范

- 视觉可以有“兵书路线图”感，但不做复杂技能树迷宫。
- 不使用黑金养成、VIP、战力飙升、红点堆叠等页游养成语言。
- 保持浅色石瓷/宣纸质感、细边框、墨色线条和克制朱红/金铜点缀。
- 明确展示“弱成长：不替代技能与装备构筑”。

## 后续出图注意

- 如果后续扩展更多节点，优先在四条路线下增加子节点密度，而不是新增大量页签。
- 若某节点未来需要前置条件，必须在文档和 UI 中明确写出，不通过默认连线暗示。
- 不要把兵书成长做成付费 4 人位或强成长入口。
