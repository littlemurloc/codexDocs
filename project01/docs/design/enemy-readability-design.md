# Enemy Readability Design

> **P0 范围更新（2026-07-25）**：本文的祭酒、渠帅、西凉及 Boss 表现为历史读性骨架，不得作为当前首轮原型依据。P0 仅可制作步卒、弓手、作为队长的力士，以及已有三国英雄的 NPC 队长变体；单位范围和数值以 [p0-npc-roster-and-scaling-v0.1.md](p0-npc-roster-and-scaling-v0.1.md) 为准。

本文件承载敌人的外观读性、技能前摇、战斗表现和 Boss 阶段提示。敌人机制见 [enemy-boss-design.md](enemy-boss-design.md)，波次配置见 [stage-wave-table.md](stage-wave-table.md)。

## 设计目标

敌人表现必须让玩家在自动战斗中快速理解 4 件事：

- 谁是普通杂兵，谁是高威胁单位。
- 当前受到的压力来自前排、远程、破甲、控制、持续伤害还是增益。
- Boss 正在准备什么技能，玩家为什么被打穿。
- 玩家下次应该调整技能、装备、站位顺序还是阵容组合。

MVP 不追求复杂演出，优先做到“第一眼能分辨，第二局能记住”。

## 通用视觉规则

敌人职责读性：

| 职责 | 外观轮廓 | 动作节奏 | 表现重点 |
| --- | --- | --- | --- |
| 杂兵 | 小体型、低装甲、成组出现 | 快速但轻量 | 数量感，不抢视觉中心 |
| 前排 | 大盾、重甲、宽站姿 | 慢速、沉重 | 拖线和高耐久 |
| 远程 | 弓、弩、火箭、后撤动作 | 攻击前摇清楚 | 让玩家知道伤害来自后方 |
| 破甲 | 枪、钩、碎甲特效 | 攻击前有明显蓄力 | 命中后出现破甲标识 |
| 控制 | 法器、战鼓、号角或特殊光效 | 技能前摇偏长 | 给玩家可预判窗口 |
| 增益 | 旗帜、鼓、令牌、军师姿态 | 周期性施法 | 光环范围和增益对象清楚 |
| 精英 | 更大体型、描边或特殊底圈 | 动作更有重量 | 和普通单位区分明显 |
| Boss | 独立模型或显著放大 | 分阶段变化 | 技能、阶段、弱点窗口都可见 |

关键词颜色建议：

| 关键词 | 主色 | 表现建议 |
| --- | --- | --- |
| 流血 | 暗红 | 持续滴血、短促伤害跳字、目标身上裂纹 |
| 灼烧 | 橙红 | 火焰、烟、燃烧边缘、范围地面提示 |
| 护盾 | 青蓝 | 半透明护罩、盾碎粒子 |
| 反击 | 金黄 | 受击后瞬间闪光和回击轨迹 |
| 破甲 | 灰白 | 护甲碎片、裂纹图标、短暂防御降低标识 |
| 连击 | 银白 | 连续残影或多段命中线 |
| 士气 | 金色 | 向上脉冲、旗帜光环 |
| 召唤 | 紫黑或土黄 | 入场法阵、尘土或符印 |
| 眩晕 | 蓝紫 | 头顶星标或定身环 |
| 治疗 | 绿色 | 上升粒子和生命恢复跳字 |

## 普通敌人表现

### 黄巾余烬

| 敌人 | 外观读性 | 技能前摇 | 受击/死亡反馈 |
| --- | --- | --- | --- |
| 黄巾步卒 | 黄布头巾、小刀、小体型 | 无明显技能前摇，以普攻为主 | 死亡快速倒地，强调成片清杂 |
| 黄巾弓手 | 背弓、站位靠后、拉弓姿势明显 | 拉弓抬手后射击 | 被突进或范围命中时后仰明显 |
| 黄巾力士 | 大体型、双手重武器、慢步 | 重击前举武器蓄力 | 破甲后出现碎甲反馈 |
| 黄巾祭酒 | 持符或铃，身边有小型符光 | 施法前抬手，脚下出现黄光圈 | 死亡后周围增益光环立刻消失 |
| 黄巾渠帅 | 更高体型、旗帜或法杖 | 召唤前地面出现符阵 | 召唤物被清后进入短暂虚弱表现 |

### 汜水先锋

| 敌人 | 外观读性 | 技能前摇 | 受击/死亡反馈 |
| --- | --- | --- | --- |
| 西凉刀兵 | 皮甲、弯刀、成组冲锋 | 普攻前小幅挥刀 | 被反击时有明显硬直 |
| 西凉盾兵 | 大盾、宽体型、前排站姿 | 举盾时盾面发光 | 盾破时出现碎片和短暂停顿 |
| 西凉弓骑 | 弓骑剪影、后撤动作 | 射击前后拉距离 | 被控场时停止后撤 |
| 破甲校尉 | 长枪或钩镰，武器带白光 | 破甲刺击前蓄力 0.6-1 秒 | 命中目标后出现破甲裂纹 |
| 汜水先锋 | 重甲先锋、冲锋姿态 | 冲阵前压低身体并出现路线提示 | 低血暴怒时模型泛红、攻速提高 |

### 赤壁火势

| 敌人 | 外观读性 | 技能前摇 | 受击/死亡反馈 |
| --- | --- | --- | --- |
| 江面轻舟 | 小舟或轻甲水兵，数量多 | 无明显技能前摇 | 死亡可有小范围水花或烟火 |
| 火箭斥候 | 火箭箭袋、箭头微光 | 点火后再射击 | 被击杀后火焰立即熄灭 |
| 连环船阵 | 盾船或连环锁链标识 | 连接时单位间出现链线 | 链接被打破时链线断裂 |
| 纵火军师 | 火扇、卷轴或火盆 | 施法前地面升起火纹 | 死亡后敌方火焰增益消失 |
| 赤壁火魁 | 火焰大型单位，脚下火场 | 大招前火场收束到 Boss 周围 | 阶段切换时火焰颜色和强度变化 |

### 虎牢压阵

| 敌人 | 外观读性 | 技能前摇 | 受击/死亡反馈 |
| --- | --- | --- | --- |
| 陷阵枪兵 | 长枪方阵、重甲、整齐队形 | 枪阵突刺前枪尖发光 | 被破甲时阵型散开 |
| 虎牢重骑 | 骑兵或重甲突进轮廓 | 冲锋前短暂停顿并出现冲锋线 | 被眩晕打断时冲锋线消失 |
| 后阵弩手 | 大弩、站位靠后、装填动作 | 装填后射击破甲目标 | 被突进时有明显后撤失败动作 |
| 战鼓督军 | 背鼓或持旗，身边有士气光环 | 敲鼓前抬槌，光环扩张 | 死亡后鼓声停止，光环消散 |
| 虎牢守将 | 重甲大体型、霸体底圈 | 霸体前身上出现护甲纹路 | 霸体被破后短暂跪地 |
| 虎牢魔将 | 终局 Boss，巨型武器和压迫站姿 | 阶段技能都有地面提示 | 阶段变化时武器、底圈、音效变化 |

## Boss 表现规则

Boss 需要同时表现 3 类信息：

- 当前阶段。
- 即将释放的技能。
- 玩家可利用的弱点窗口。

Boss 阶段表现建议：

| Boss | 阶段/状态 | 表现方式 | 玩家理解 |
| --- | --- | --- | --- |
| 黄巾渠帅 | 聚众 | 地面符阵亮起，周围出现召唤预告点 | 即将召唤杂兵 |
| 黄巾渠帅 | 溃散窗口 | 旗帜下垂，身上黄光熄灭 | 可以集中攻击本体 |
| 汜水先锋 | 冲阵 | 地面出现直线路径，Boss 压低身体 | 即将冲击前排 |
| 汜水先锋 | 铁壁 | 护盾纹路覆盖身体 | 需要破盾或等窗口 |
| 汜水先锋 | 暴怒 | 红光、呼吸加重、攻击动作加快 | 低血危险，但也是终结窗口 |
| 赤壁火魁 | 蔓延火场 | 地面火纹从 Boss 向外扩散 | 需要清火舟或治疗抗压 |
| 赤壁火魁 | 余烬爆发 | 火焰回流到 Boss 身上再炸开 | 需要控场或尽快处理灼烧源 |
| 虎牢魔将 | 压阵 | 黑红底圈，召唤枪兵预告 | 第一阶段偏拖线 |
| 虎牢魔将 | 破军 | 武器发白光，攻击附破甲裂纹 | 前排正在被打穿 |
| 虎牢魔将 | 无双 | 红黑光效，动作加快，弱点窗口闪烁 | 高危终结阶段 |

## 技能前摇规范

普通敌人：

- 普攻前摇可以较短，保持战斗节奏。
- 功能技能必须有 0.5-1 秒可读前摇。
- 破甲、控制、增益类技能需要比普通伤害技能更清楚。
- 远程攻击必须能从弹道或命中反馈看出来源。

精英与 Boss：

- 关键技能必须有地面提示、动作前摇或状态变化之一。
- 大范围技能需要提前显示范围。
- 召唤技能需要提前显示召唤点。
- 阶段切换需要有短暂停顿或明显音画变化。
- 不使用无前摇瞬发强控、无提示大伤害和不可解释的秒杀。

## UI 与状态提示

敌人头顶或脚下建议表现：

- 精英：特殊底圈或小型精英标识。
- Boss：独立 Boss 血条、阶段标识、技能蓄力条。
- 破甲：护甲裂纹图标，持续时间用淡出表现。
- 灼烧/流血：小图标 + 层数或强度提示。
- 眩晕：头顶控制图标，单位动作停止。
- 增益光环：脚下范围圈，增益来源死亡后立即消失。

MVP 可先做轻量提示，不必做复杂 UI 面板。重点是战斗中玩家能判断“谁需要先杀”。

## 关卡表现重点

| 关卡 | 表现主题 | 读性重点 |
| --- | --- | --- |
| 黄巾余烬 | 黄布、符令、杂兵潮 | 新手能看懂杂兵、弓手、力士、祭酒的职责差异 |
| 汜水先锋 | 重甲、盾阵、冲锋 | 破甲和护盾变化必须清楚 |
| 赤壁火势 | 火场、轻舟、连环 | 灼烧范围、扩散、火场危险区必须清楚 |
| 虎牢压阵 | 重骑、枪阵、战鼓 | 增益来源、冲锋路径、Boss 阶段必须清楚 |

## 验收标准

- 玩家第一次见到远程敌人时，能通过站位和攻击动作判断其威胁来源。
- 玩家第一次被破甲时，能从目标状态上看出防御被削弱。
- 玩家能在 1-2 次战斗内理解增益兵需要优先击杀。
- Boss 释放关键技能前至少有一种清楚预告。
- Boss 进入弱点窗口时有明显状态变化。
- 同屏单位较多时，精英和 Boss 不会被杂兵视觉淹没。
- 状态颜色与关键词保持一致，不频繁复用相互冲突的颜色。
