- 所有 Set、卡牌和军资均为单局内容；本局结束清空。

## 全局执行口径

- 主动军令每波最多使用 1 次，消耗 1 点军令点；普通/精英波 2 点，Boss 波 3 点。
- 每次备战从军令库选择至多 3 张主动军令进入 HUD。
- “上、中、下路”均为战前路归属。敌方上、中、下路为 `147 / 258 / 369`；队长与小兵的归属在本波固定。
- “该队伍”指同一战前格位中的队长及其存活小兵；队长或小兵在战斗中移动不改变其队伍归属。
- 灼烧不叠加，保留较高的每秒伤害并按主规则刷新持续时间。
- 主动军令预警期间若目标或目标路完全失效，且卡牌写明可返还，则不消耗军令点并可在本波重新使用。

## 破军 I

| 卡牌 | 类型 | 效果 |
| --- | --- | --- |
| 锋矢军略 | 被动 | 全部友方队长攻击提高 8%；其小兵攻击提高 4%。 |
| 战鼓军略 | 被动 | 每当敌方队长或精英死亡时，全部存活友方队长及小兵攻击速度提高 4%，最多叠加 3 层。本波内生效。普通小兵死亡不触发。 |
| 斩将令 | 主动 | 选择一名敌方队长或精英。0.6 秒预警后，连续造成 3 次伤害，每次为目标最大生命值 2% 的真实伤害，共 6%。目标在预警期死亡或离场则返还本次使用；目标在三次伤害中死亡，剩余伤害取消且不改选目标。 |

### 自动结果：破阵令

- 类型：主动进阶；斩将令自动替换为破阵令，保留原军令库/HUD 槽位；锋矢军略与战鼓军略继续生效。
- 效果：选择敌方战前上、中、下任一路。0.8 秒预警后，该路全部存活敌方单位各受到自身最大生命值 4% 的真实伤害；该路队长与精英额外受到自身最大生命值 2% 的真实伤害。全部命中单位防御降低 15%，持续 4 秒。
- 目标路在结算前已无存活敌人时返还本次使用。表现可出现击退，但只改变实时战斗坐标，不改变战前路、行列或站位属性。
- 解锁：虎贲 II、飞骑 II。

### 自动结果：焚舟令

- 类型：主动进阶；连环令自动替换，保留两张被动效果。
- 效果：选择敌方战前上、中、下任一路；按相同规则锁定该路生命比例最低的存活敌方队长或精英。0.8 秒预警后，对其造成最大生命值 6% 的真实伤害，施加持续 5 秒、每秒最大生命值 1.5% 真实伤害的灼烧，并施加持续 8 秒的火媒。
- 火媒期间目标死亡时，相邻敌路全部存活单位各受到最大生命值 4% 的真实伤害，且防御降低 12%，持续 4 秒。连锁伤害不附加灼烧。
- 目标在预警阶段死亡或离场则返还本次使用。

## 表现与 UI 约束

- 主动军令选路时显示战前路归属和最终自动锁定目标；不得以实时地面区域代替战前路。
- Set 组件显示 Set 名称、图标、当前进度、已拥有组件、缺失组件和自动结果；二阶组件显示来源高阶结果与“新解锁”状态。
- 融合发生时，明确展示“基础主动已进阶”或“被动已吞噬至军略图鉴”，并列出继承效果。
- 飞骑、伏兵等军令使用临时战术表现，不调度已有英雄队伍移动。
