# 军令卡系统 V1.1

> 状态：当前主规则源。本文替代 V1 中“高阶卡解锁后再进入报价购买”的旧表述；具体卡牌效果以 [tactical-command-card-catalog-v1.md](tactical-command-card-catalog-v1.md) 为准。

## 设计定位

英雄已解锁主动技能始终自动释放，保持英雄与其小队的稳定职业循环。军令卡承担局内构筑、备战取舍和战斗中的战术介入；军令不替代英雄技能，也不把英雄主动技能改造成手动卡牌。

所有军令卡、军资、报价资格、Set 进度、融合结果和属性收益只在单局内存在，单局结束后全部清空。

## 卡牌类型与持有规则

- 军令卡只分为主动军令和被动军略。
- 主动军令进入局内军令库；每次备战可从已拥有主动军令中选择至多 3 张进入下一波 HUD。
- 被动军略购买后在本局后续波次持续生效，收纳在军略图鉴，不占 HUD 槽位。
- Set 组件和独立军略本局最多持有 1 次；已拥有的卡不会重复报价。
- 高阶结果不是第三类卡，而是组件完成 Set 后自动获得的进阶形态；高阶结果不进入报价市场，也不额外消耗军资。

## 主动军令与军令点

- 普通波与精英波每波获得 2 点军令点；Boss 波获得 3 点。
- 每张已备战的主动军令每波最多使用 1 次，默认消耗 1 点军令点。
- 波末未使用的军令点清空，不跨波储存。
- 主动军令最多允许一次手动目标选择。卡牌可无目标、选择一名合法单位、一路战前路归属或一片明确区域；不得要求玩家连续进行两次目标选择。
- 卡牌预警期间目标失效、且卡牌规则声明可返还时，不消耗军令点，该卡本波可重新使用。具体合法目标与返还条件以卡牌目录为准。

## 战前路归属与目标规则

- 军令中的“敌方上、中、下路”或“我方上、中、下路”，均指开战前的战前路归属，不指战斗中的实时地图坐标。
- 敌方上、中、下路分别对应战前 `147 / 258 / 369`；我方按其战前布阵映射记录相同的上、中、下路归属。
- 队长及其小兵继承同一战前路归属；移动、追击、后撤、击退或跨路战斗不会改变归属。
- 涉及“后排优先、中排优先、前排优先”的军令，按目标队伍的战前列归属判定，不读取实时坐标。
- 需要系统自动锁定对象时，必须在 UI 中预览最终目标，避免玩家对选路结果产生误解。

## 备战军令市场

- 每次备战展示 3 个军令报价。玩家可在同一次备战中购买多张已报价卡，也可支付军资刷新整组报价。
- 报价由路线位、探索位和自由位构成：路线位优先服务玩家已开始但未完成的 Set；探索位服务未开始根系、独立军略和已解锁的二阶路线；自由位从当前全部合法候选中产生。
- 开局报价池包含三条根系的 9 张组件，以及鼓舞令、固守令、体魄军略、锐气军略。第 3 波起，行军军略和保全军略加入独立军略池。
- 根系完成并获得高阶结果后，对应两条二阶路线的 6 张组件进入候选池。后续连续两次备战报价中，各保证出现 1 张“新解锁”组件，分别来自这两条二阶路线，顺序随机；卡牌仍需正常购买。
- 刷新后的具体价格曲线、基础卡价、初始军资、波次收入和报价权重属于经济核算项，尚未定稿。

## Set、吞噬与自动进阶

- “2 张被动 + 1 张基础主动”的 Set：两张被动效果继续生效并折叠入高阶说明，基础主动自动进阶为高阶主动，保留原有军令库与 HUD 槽位。
- “3 张被动”的 Set：三张效果继续生效并自动吞噬为一个后台高阶被动，展示在军略图鉴中。
- 自动融合不应让玩家感觉已购买收益消失；高阶说明必须列出继承效果和新增效果。
- 根系高阶结果可一对多解锁后续二阶 Set；解锁只赋予报价资格，不免费赠送组件。
- 第一版图谱只包含根系与二阶路线，不继续进入第三层。

## 通用状态与效果结算

### 灼烧

- 灼烧每秒结算 1 次，伤害以卡牌说明的“目标最大生命值百分比”计算，伤害类型为真实伤害。
- 同一单位同时受到多个灼烧时不叠加。新灼烧的每秒伤害更高时，替换现有效果并使用新持续时间；每秒伤害相同或更低时，仅把现有效果的剩余时间延长至两者较长值，不降低现有伤害档位。

### 护盾与属性修正

- 护盾以目标自身最大生命值百分比计算，持续时间以卡牌说明为准。
- 攻击、最大生命值、防御、攻击速度、移动速度、闪避、暴击、造成伤害和受到伤害等修正均应作为独立可配置 Modifier 结算；其最终叠加公式和舍入规则由战斗数值核算统一确定。

### 小兵补员

- 小兵阵亡后跨波次保留损失，波次结束不会自动补全。
- 只有明确写出补员效果的军略可在备战阶段补充小兵；补员对象必须是仍存活的玩家队长，且小兵数量不得超过该队长当前升星对应的上限。
- 补充的小兵继承队长预设兵种；当前兵种仅为近战或远程。

## 设计约束

- 主动军令应制造可观察、可判断时机的战场事件，避免大量同质化的短时数值按钮。
- 军令表现可召来飞骑、伏兵、箭阵等临时战术效果，但不得让既有英雄队伍脱离自身自动战斗逻辑，也不得改变其战前站位属性。
- 被动军略允许提供整局属性、经济、成长、兵种、站位或战场规则收益，不限制为短时 Buff。
- 军令不得绕过英雄升星解锁技能的规则，不得替代英雄自动技能的职业身份。

## 当前待经济与数值核算项

- 初始军资、每类波次基础军资收入、基础卡价和刷新价格曲线。
- 报价权重、保底算法、购买后的补位规则和每次备战刷新上限。
- 暴击、攻速和移速的完整公式、叠加顺序与舍入规则。物理防御公式、伤害加减区间、英雄 1 至 6 级成长与技能固定 1 级已由 [p0-attribute-progression-v0.1.md](p0-attribute-progression-v0.1.md) 和 [unit-attribute-design.md](unit-attribute-design.md) 定义。
- 英雄经验的逐级显式数值表与波次经验预算；不得影响英雄升星解锁技能的既定边界。
