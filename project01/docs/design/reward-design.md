# Reward Design

当前奖励系统只保留难度通关奖励、首通解锁和局外资源。旧版装备奖励、技能强化、战备重掷暂不作为当前奖励要求。

## 奖励触发边界

- 普通波胜利：只进入备战，不发放通关奖励。
- 最后一波胜利：判定当前难度通关，展示通关奖励并结算局外资源。
- 失败：进入失败结算，不展示通关奖励。

## 当前奖励类型

| 类型 | 当前状态 | 说明 |
| --- | --- | --- |
| 局外资源 | 保留 | 用于后续兵书或英雄成长。 |
| 首通解锁 | 保留 | 可解锁新关卡、新难度、英雄候选或主题内容。 |
| 通关奖励 | 保留 | 可使用 3 选 1 展示形式，但内容暂以局外资源和解锁项为主。 |
| 装备奖励 | 暂缓 | 旧版装备已废弃。 |
| 技能强化 | 暂缓 | 当前技能通过升星解锁。 |
| 战备重掷 | 暂缓 | 旧版战备已废弃。 |
