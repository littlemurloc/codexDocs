# Battle Prep Choice Design

本文件定义局内“战备抉择玩法”。它替代自走棋式局内商店购买英雄的设计方向，成为每波战斗结束后的主要局内成长系统。

核心概念：每一波次战斗结束后，游戏进入【备战状态】。玩家先预览下一波敌人队伍信息，再从随机出现的 3 个战备能力中选择 1 个。被选择的战备能力会在本次挑战内持续继承，直到当前挑战结束；新开一局时从零重新积累。

## 设计目标

- 用“看下一波敌人 -> 做战前抉择”强化策略感。
- 用 3 选 1 Roguelike 能力池替代局内买英雄商店。
- 让每波战斗之间都有短促但有意义的成长节点。
- 让玩家围绕敌人情报、当前阵容、技能装配、装备和核心流派做判断。
- 保持单局 15-25 分钟节奏，避免准备阶段变得冗长。
- 能力只在本次挑战内继承，不形成局外永久战力。

## 基础流程

1. 当前波次战斗结束。
2. 游戏暂停，进入【备战状态】。
3. 系统展示下一波敌人队伍预览。
4. 系统从战备能力池中随机抽取 3 个候选。
5. 玩家根据敌人信息、当前阵容和已有能力选择 1 个。
6. 被选择能力加入本局战备能力列表，并持续生效到当前挑战结束。
7. 玩家确认出战顺序后进入下一波战斗；技能装配和装备默认不可自由更换，除非获得特定战备能力或奖励授权。

战备抉择不负责购买武将。武将阵容更偏向开局选择、关卡奖励、局外解锁或预设队伍方案；波次间成长主要由战备能力承担。

## 下一波敌人预览

敌人预览不是完整数值透出，而是给玩家足够做判断的信息。

建议展示：

| 信息 | 展示方式 | 设计目的 |
| --- | --- | --- |
| 敌人类型 | 图标 + 名称 + 数量段 | 判断是杂兵、前排、远程、破甲、控制、增益还是 Boss |
| 核心威胁 | 1-2 个标签 | 例如高护甲、灼烧、后排火力、召唤、冲锋 |
| 关键词 | 状态图标 | 提前知道需要护盾、治疗、破甲、清场或点杀 |
| 波次节奏 | 慢速、标准、密集、压迫、Boss | 判断要选即时能力还是长期能力 |
| 推荐应对 | 简短标签 | 例如范围、单体、续航、控场、后排处理 |

不建议展示：

- 每个敌人的完整数值。
- 复杂公式。
- 过多弱点提示导致玩家只按推荐答案选择。

## 战备能力基础规则

| 规则 | 说明 |
| --- | --- |
| 出现时机 | 每波战斗结束后的备战状态 |
| 选择形式 | 随机 3 选 1 |
| 生效范围 | 本次挑战持续生效 |
| 重置规则 | 挑战结束后清空，新局从零开始 |
| 叠加规则 | 同名能力可禁止重复，也可设计为可升级 |
| 稀有度 | MVP 建议普通、稀有、史诗 3 档 |
| 选择耗时 | 单次目标 15-40 秒 |

MVP 建议：

- 每次只选 1 个能力。
- 不提供基础刷新按钮，避免玩家变成刷池玩法。
- 可以把“重掷一次 3 选 1”作为少量奖励或兵书节点，而不是基础规则。
- 同名能力默认不重复出现；若设计为可升级，应明确等级上限。

## 备战状态权限

为了保留 6 选 4 技能装配和装备配置的战前重量，MVP 默认备战状态只允许有限调整：

| 操作 | 默认是否允许 | 说明 |
| --- | --- | --- |
| 查看下一波敌人预览 | 是 | 备战抉择的核心信息来源。 |
| 选择战备 3 选 1 | 是 | 每波结束后的主要成长节点。 |
| 查看已选战备能力 | 是 | 帮助玩家理解当前构筑方向。 |
| 调整出战顺序 | 是 | 允许针对下一波远程、冲锋或功能怪做轻量应对。 |
| 调整技能装配 | 默认否 | 需要 `战术变招` 等能力授权，避免每波都变成免费换技能。 |
| 更换装备 | 默认否 | 只在挑战开始前配置；若后续加入特殊能力，应限制次数和范围。 |

这样处理后，战前技能和装备选择仍然重要，波次间抉择则集中在敌人预览、战备能力和出战顺序上。

## 能力池维度

战备能力池需要覆盖多种策略方向，而不是只加攻击或生命。

### 1. 关键词强化

强化玩家当前构筑的核心关键词。

| 能力方向 | 示例能力 | 策略价值 |
| --- | --- | --- |
| 流血 | 流血层数上限 +1；攻击流血目标时小幅提高斩杀伤害 | 强化流血斩杀，适合精英和 Boss |
| 灼烧 | 灼烧持续时间提高；灼烧目标死亡时小范围传火 | 强化清杂和密集波次 |
| 护盾 | 战斗开始时前排获得护盾；护盾破裂后短暂减伤 | 强化正面抗压 |
| 反击 | 拥有护盾时反击伤害提高 | 连接护盾反击流派 |
| 破甲 | 首次命中精英或 Boss 时施加短时破甲 | 应对厚前排和高护甲 |
| 连击 | 连击最后一击附带低倍率额外伤害 | 强化敏将输出 |
| 士气 | 每波开始获得少量士气 | 加速技能循环和辅将价值 |

设计边界：

- 关键词强化应优先服务已携带技能或已装备方向。
- 不应出现“无条件全伤害大幅提高”这种万能能力。

### 2. 波次针对

根据下一波敌人信息做针对选择。

| 敌方压力 | 示例能力 | 适合选择时机 |
| --- | --- | --- |
| 大量杂兵 | 开战前 8 秒范围技能伤害提高 | 下一波密集小怪 |
| 厚前排 | 前 10 秒对护甲高的敌人造成额外破甲 | 盾兵、守将、Boss 前排 |
| 后排远程 | 我方敏将首次攻击优先锁定远程单位 | 弓手、火箭斥候、后阵弩手 |
| 破甲爆发 | 我方前排首次被破甲时获得护盾 | 破甲校尉、虎牢魔将 |
| 持续灼烧 | 每波首次受到灼烧时获得短时治疗增益 | 赤壁火势 |
| 敌方增益 | 对增益单位的首次伤害提高 | 祭酒、战鼓督军 |
| Boss | Boss 进入弱点窗口时我方技能冷却小幅缩短 | Boss 波 |

设计边界：

- 针对能力可以很强，但适用面要窄。
- 玩家应能通过预览信息判断“这次选它很赚”。

### 3. 开战布置

改变下一波开始时的战斗节奏。

| 能力 | 效果方向 | 乐趣点 |
| --- | --- | --- |
| 先军列阵 | 战斗开始前排获得护盾 | 稳住开局冲锋 |
| 火攻预备 | 第一名智将开场技能冷却缩短 | 灼烧队更快启动 |
| 伏击后阵 | 第一名敏将开场优先攻击后排 | 处理远程和功能怪 |
| 固守阵脚 | 前 6 秒我方不被击退或减速 | 应对压迫波 |
| 鼓舞开阵 | 开局获得少量士气 | 支持技能流和辅将 |
| 临阵疗伤 | 每波开始恢复少量生命 | 支持长难度多波续航 |

设计边界：

- 开战能力适合新手理解，表现要直接。
- 不能让开局能力直接清掉一整波，否则会削弱战斗过程。

### 4. 角色职责强化

强化某类职业或站位，让阵容差异更明显。

| 强化对象 | 示例能力 | 目的 |
| --- | --- | --- |
| 猛将 | 猛将受到近战伤害降低 | 强化前排承伤 |
| 智将 | 智将首次释放技能后提高灼烧或控制效果 | 强化技能爆发 |
| 敏将 | 敏将攻击低血目标时获得攻速 | 强化收割 |
| 辅将 | 辅将治疗或士气效果提高 | 提高辅助价值 |
| 前排 | 最靠前单位获得额外护盾 | 解决单线接敌压力 |
| 后排 | 最靠后单位受到远程伤害降低 | 保护输出 |

设计边界：

- 职业强化要让不同阵容有不同答案。
- 不要让某个职业强化永远优于关键词强化。

### 5. 技能装配联动

让玩家的 6 选 4 技能装配和战备能力互相影响。

| 能力 | 效果方向 | 策略点 |
| --- | --- | --- |
| 专精打磨 | 随机强化一名武将已携带技能的关键词 | 鼓励围绕当前装配选 |
| 空槽补强 | 携带治疗或护盾技能的武将获得额外生存 | 支持防守装配 |
| 连携施法 | 两名武将拥有相同关键词时，相关技能冷却小幅降低 | 鼓励队伍同主题 |
| 战术变招 | 本次备战允许调整一名武将技能装配 | 面对下一波敌人临时变阵 |
| 技能预热 | 下一波第一轮技能冷却缩短，但后续恢复正常 | 针对压迫开局 |

设计边界：

- 不要让战备能力替代技能装配本身。
- 能力应鼓励玩家理解“我带了哪些技能”，而不是忽略技能池。

### 6. 装备联动

放大已有装备的流派特征。

| 能力 | 效果方向 | 适配 |
| --- | --- | --- |
| 盾纹加固 | 护盾类装备效果提高 | 护盾反击 |
| 火器整备 | 灼烧类装备首次触发效果提高 | 灼烧扩散 |
| 利刃淬血 | 流血类装备对精英收益提高 | 流血斩杀 |
| 轻装追击 | 连击类装备追加命中伤害提高但不复制强效果 | 敏将连击 |
| 军械保养 | 每波开始随机强化一件已装备道具的小效果 | 混合阵容 |

设计边界：

- 装备联动必须依赖已装备道具，不能凭空给万能强度。
- 高风险装备如 `白马镫`、`还血护符`、`断首符` 需要专门限制叠加。

### 7. 风险收益

提供更刺激的选择，但要避免成为唯一最优。

| 能力 | 正面 | 代价 |
| --- | --- | --- |
| 破釜沉舟 | 我方低血时伤害提高 | 治疗效果降低 |
| 急行军 | 开局攻速提高 | 前 5 秒受到伤害提高 |
| 猛火计 | 灼烧伤害提高 | 我方护盾效果降低 |
| 血战到底 | 流血和斩杀提高 | 战斗结束后恢复效果降低 |
| 孤注一击 | 下一波奖励提高 | 下一波敌人攻击提高 |

设计边界：

- 风险收益能力不宜过多，MVP 可占能力池 10%-15%。
- 代价必须清楚显示，避免玩家误选后觉得被坑。

### 8. 资源与修整

提供恢复、重掷、临时调整等非直接战力选项。

| 能力 | 效果方向 | 用途 |
| --- | --- | --- |
| 整军休息 | 全队恢复生命 | 多波长难度续航 |
| 重新侦察 | 下次战备抉择可重掷一次 | 提高构筑稳定性 |
| 临时军械 | 下一波强化一件已装备道具的小效果 | 补流派缺口，但不新增装备栏或开放自由换装 |
| 军令调度 | 本次备战可额外调整一次站位锚点或前后排保护优先级 | 针对后排、冲锋或功能怪，区别于默认出战顺序调整 |
| 战术回收 | 移除一个低价值战备能力，换取一次强化候选 | 后期修正构筑 |

设计边界：

- 资源类能力要少量出现，避免玩家总选保守项。
- 重掷和移除能力要受次数限制。

## 稀有度与能力数量

MVP 首批建议先做 30 个战备能力，具体清单见 [battle-prep-choice-pool.md](battle-prep-choice-pool.md)。30 个是第一版验证规模，后续根据测试结果再扩展到 45 个以上。

| 稀有度 | 占比 | 设计方向 |
| --- | --- | --- |
| 普通 | 60% | 基础关键词、开战布置、职业小强化 |
| 稀有 | 30% | 明显改变节奏的波次针对、装备联动 |
| 史诗 | 10% | 强流派核心、风险收益、Boss 针对 |

首版 30 个能力池建议：

- 关键词强化：8 个。
- 波次针对：6 个。
- 开战布置：4 个。
- 职业职责：4 个。
- 技能装配联动：3 个。
- 装备联动：3 个。
- 风险收益：1 个。
- 资源修整：1 个。

## 随机规则

一次 3 选 1 不应出现三个高度相似选项。

建议规则：

- 同一次候选中，尽量覆盖至少 2 个不同维度。
- 如果下一波是 Boss，候选池提高 Boss 针对、单体爆发、续航类权重。
- 如果下一波是密集小怪，候选池提高范围、灼烧、反击类权重。
- 如果玩家已有某关键词能力，可适度提高同流派后续能力出现率，但不能锁死。
- 同名能力默认不重复；可升级能力需要显示当前等级和上限。

## 与敌人预览的联动

战备抉择的乐趣来自“我知道下一波大概是什么，所以我选择这个”。

示例：

| 下一波预览 | 玩家可能选择 | 形成的判断 |
| --- | --- | --- |
| 盾兵 + 破甲校尉 | 首次命中高护甲敌人施加破甲 | 先破前排，避免被拖死 |
| 火箭斥候 + 轻舟密集 | 灼烧扩散或开场范围强化 | 先清杂，再压后排 |
| 战鼓督军 + 重骑 | 敏将优先锁定增益单位 | 先切督军，降低整波压力 |
| Boss 波 | 弱点窗口技能冷却缩短 | 把爆发留给窗口 |
| 后阵弩手群 | 后排减伤或敏将伏击 | 防止远程消耗打穿 |

## 与现有系统的关系

| 系统 | 与战备抉择的关系 |
| --- | --- |
| 武将 | 开局或局外决定基础队伍，战备能力强化其战斗方向 |
| 技能装配 | 战备能力根据已携带技能关键词形成更强分支 |
| 装备 | 装备提供角色风格，战备能力可进一步放大装备关键词 |
| 羁绊 | 羁绊决定队伍倾向，战备能力提供每局变化 |
| 关卡波次 | 下一波敌人预览决定战备能力选择价值 |
| 局外成长 | 可提供少量重掷、候选质量或初始能力，但不能直接给强战力 |

## UI 信息结构

备战状态建议分为三块：

1. 下一波敌人预览：敌人类型、数量段、威胁标签、波次节奏。
2. 战备 3 选 1：能力名、稀有度、效果、适配关键词、是否当前挑战持续。
3. 当前战备列表：已获得能力、核心关键词、可叠加状态。

交互要求：

- 玩家选择能力前可以查看已有能力列表。
- 每个候选能力要显示“适合应对什么”。
- 有代价的能力必须用明显提示展示代价。
- 选择后进入下一波前，玩家默认可调整出战顺序；技能装配或装备调整只在获得对应授权时开放。

## 首轮测试关注

- 玩家是否能理解“下一波预览”和“3 选 1能力”的关系。
- 玩家是否会根据敌人信息改变选择，而不是永远选最高稀有度。
- 三条核心流派是否都能通过战备能力获得成长路线。
- 能力是否过于数值化，导致选择无聊。
- 能力是否过于复杂，导致备战阶段拖慢局长。
- 本局继承能力是否让后期滚雪球过强。

## 验收标准

- 每波战斗结束后能进入备战状态。
- 备战状态能预览下一波敌人队伍信息。
- 每次战备抉择提供 3 个随机能力，玩家选择其中 1 个。
- 已选择能力在本次挑战中持续继承，直到当前挑战结束。
- 新开一局时，已选战备能力清空。
- 能力池至少覆盖关键词强化、波次针对、开战布置、职业职责、技能联动、装备联动、风险收益、资源修整 8 个维度。
- 玩家能根据下一波敌人信息做出不同选择。
- 战备能力不替代技能装配、装备和羁绊，而是放大它们。
