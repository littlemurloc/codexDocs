# P0 技能数值基线 v0.1

> 状态：P0 提案。用于首批 10 英雄的 60 个技能首次落数值；不继承旧版装备、战备、羁绊或绝对战力基线。

## 目标

- 先让 10 名英雄在同一尺度上可比较、可实现、可测试。
- 以相对系数落表，暂不绑定关卡、装备或局外成长后的绝对面板。
- 英雄主动技能始终自动释放。军令卡系统的具体数值、主动释放限制与属性换算以 [tactical-command-card-system-v1.md](tactical-command-card-system-v1.md) 后续卡牌设计为准，当前不在本数值基线中预设。
- 数值首先服务单人 PVE 的 3x3 小队战斗：普通波次应持续约 15 至 25 秒，精英波次约 25 至 35 秒，Boss 波次约 35 至 50 秒。

## P0 基础属性口径

| 属性 | 配置字段 | P0 规则 |
| --- | --- | --- |
| 最大生命 | `max_hp` | 治疗、护盾、低生命阈值均以单位自身最大生命百分比结算。 |
| 攻击 | `attack` | 物理与法术技能的基础系数统一引用施法者 `attack`；法术 / 物理仅作为伤害标签和后续扩展入口。 |
| 攻击速度 | `attack_speed` | 以普攻间隔或每秒攻击次数实现；技能中的攻速变化使用百分比乘区。 |
| 攻击距离 | `attack_range` | 普攻与“当前目标”主动技能的默认可施放距离。 |
| 闪避 | `dodge_pct` | 成功闪避时不受本次伤害；赵云游龙等效果以此事件触发。 |
| 伤害标签 | `damage_tag` | `physical` / `spell` / `true`。P0 暂不拆物理抗性与法术抗性，标签只用于触发筛选、Boss 规则和后续扩展。 |

P0 技能先计算来源伤害：`来源伤害 = attack × coefficient`。已确认的物理伤害最终结算为：

```text
实际物理伤害
= 来源伤害 × 100 / (100 + 有效防御)
× (1 + 所有伤害加成% - 伤害减免%)
```

- 有效防御最低为 `0`；P0 暂不设计负防御。
- 防御 `100` 对应 50% 减伤；项目不采用编辑器默认的 `10000` 常数。
- 当前伤害加成与伤害减免处于同一加减区间，不能误按两个独立乘区核算。
- 法术伤害与真实伤害的最终防御/修正规则仍待独立实测，暂不得以本物理公式外推。

## 主动技能冷却档位

| 阶段 | 标准 CD | 可调整范围 | 用途 |
| --- | --- | --- | --- |
| 阶段 1 主动 | 8 秒 | 7 至 9 秒 | 基础循环、单点、低成本自保。 |
| 阶段 2 主动 | 11 秒 | 10 至 12 秒 | 范围、控制、压制或资源转换。 |
| 阶段 3 主动 | 15 秒 | 13 至 16 秒 | 高价值爆发、群体恢复、斩杀或强控制。 |

- 英雄主动技能始终自动释放；军令卡不占用或复用英雄技能 CD。
- 英雄主动技能无合法目标时不进入释放，CD 不开始。
- 通过资源强化的主动技能不额外提高 CD，强化只改变本次效果。

## 伤害系数档位

| 效果类型 | P0 推荐系数 | 说明 |
| --- | --- | --- |
| 阶段 1 单体伤害 | 160% 至 220% 攻击 | 用于基础单点、突击或附带轻控制。 |
| 阶段 2 单体伤害 | 180% 至 260% 攻击 | 需要承担压制、资源转换或更明确的条件。 |
| 阶段 2 范围伤害 | 每目标 100% 至 150% 攻击 | 以命中 3 至 5 个目标作为价值来源。 |
| 阶段 3 单体爆发 | 280% 至 360% 攻击 | 可搭配资源消耗或低生命条件。 |
| 阶段 3 范围伤害 | 每目标 150% 至 210% 攻击 | 不与强控制、满资源爆发同时取上限。 |
| 条件增伤 | 基础效果的 25% 至 80% | 仅在背刺、低生命、满资源等明确条件满足时生效。 |

- 多段攻击应按“总伤害预算”校验，不因分段而获得额外预算。
- 满资源强化优先采用额外命中、明确的系数提升或效果升级；不叠加多个无法阅读的乘区。
- 当前已确认的司马懿、甘宁数值不强行改写，后续只用本表校验其是否越界。

## 治疗、护盾与防护档位

| 效果类型 | P0 推荐值 | 说明 |
| --- | --- | --- |
| 单体治疗 | 12% 至 18% 目标最大生命 | 阶段 1 或阶段 2 单体救援。 |
| 群体治疗 | 每名队长 6% 至 10% 最大生命 | 只作用队长的群体恢复；应有较长 CD。 |
| 持续恢复 | 每秒 2% 至 3% 最大生命 | 典型持续 6 秒；总量应低于一次高价值单体治疗。 |
| 单体护盾 | 14% 至 22% 目标最大生命 | 典型持续 6 至 8 秒。 |
| 自身护盾 | 18% 至 28% 自身最大生命 | 坦克承压或低生命保命；应搭配 CD 或每波限制。 |
| 常驻减伤 | 6% 至 12% | 站位、资源层数或兵种关系提供的稳定收益。 |
| 短时减伤 | 15% 至 25% | 典型持续 3 至 6 秒，不与大护盾长期重叠。 |

## 控制、减益与移动规则

| 效果 | P0 推荐值 | 约束 |
| --- | --- | --- |
| 嘲讽 | 3 秒 | 只改变普通单位选敌；Boss 按抗性或免疫规则处理。 |
| 眩晕 | 1 秒 | 阶段 3 范围控制的上限；Boss 可缩短或免疫。 |
| 打断 | 立即生效 | 仅打断可被打断的施法，不能跳过 Boss 关键阶段。 |
| 攻击降低 | 15% 至 30% | 与攻速降低同时存在时，两者不应均取上限。 |
| 攻速降低 | 20% 至 35% | 典型持续 5 至 7 秒。 |
| 减速 | 25% 至 35% | 典型持续 3 秒；只改变战斗移动，不改变战前锁定站位属性。 |
| 击退 | 仅改变战斗坐标 | 不改变前、中、后排或侧翼属性；距离与碰撞规则由战斗程序统一实现。 |
| 隐身 | 3 至 5 秒 | 降低普通敌人选敌优先级；普攻是否破隐必须按技能单独配置。 |

## 资源与被动限制

| 类型 | P0 推荐规则 |
| --- | --- |
| 核心资源层数 | 默认上限 3；仅夏侯惇刚烈保持已确认的 4 层。 |
| 资源获取频率 | 受击、闪避、普攻、击杀等事件型来源应设置 1 秒左右的获取上限，避免攻速或多段异常放大。 |
| 低生命触发 | 默认阈值使用 35% 或 40%；每波一次或 12 秒 ICD 二选一。 |
| 击杀参与 | 默认要求目标死亡前 3 至 4 秒内造成过伤害。 |
| 小兵增益 | 只强化自然攻击、承伤或生存，不得强制小兵改选目标。 |

## 首轮落数值顺序

1. 先为每位英雄的 3 个主动技能填入 CD、伤害 / 治疗 / 护盾系数和最大目标数。
2. 再填资源强化的增幅与被动持续时间。
3. 最后填自动释放阈值、Boss 抗性与范围几何参数。
4. 每完成 1 名英雄，使用 1 名标准敌方队长、4 名小兵和 1 个精英队长进行单英雄验证；不在纸面上一次性锁死 60 个技能。

## 当前需要审定的基础决定

- P0 是否采用“物理 / 法术仅为标签、不拆双抗”的简化防御模型。
- 普通波次 25 至 40 秒、精英或 Boss 波次 45 至 75 秒的节奏目标是否合适。
- 单体治疗、护盾和阶段 CD 档位是否作为首轮统一标准。
- 军令卡系统 V1 的首批卡牌目录、局内经济与数值边界，待在英雄技能数值外单独设计。
