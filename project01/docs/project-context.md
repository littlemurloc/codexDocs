## 当前核心玩法口径

- 单人 PVE 自动战斗。
- 我方左侧 3x3 站位：`741 / 852 / 963`。
- 敌方右侧 3x3 站位：`147 / 258 / 369`。
- 前排 = `1/2/3`，中排 = `4/5/6`，后排 = `7/8/9`，侧翼 = `1/4/7` 或 `3/6/9`。
- 每名英雄带小兵队伍，队长等于英雄。
- 玩家队长死亡后小兵溃败，NPC 队长死亡后小兵继续战斗。
- 小兵阵亡跨波次保留损失，默认不自动补全；军略补员不得突破英雄升星带兵上限。
- 每名英雄 6 技能，3 主动 + 3 被动，通过升星解锁。
- 英雄进入战斗自动携带所有已解锁技能。
- 已解锁主动技能始终自动释放，不提供按英雄关闭自动释放的开关。
- 备战阶段展示 3 个军令报价，玩家使用局内军资购买军令卡并可付费刷新；同次备战可购买多张。
- 军令卡只在本局有效；主动军令在 HUD 中释放，被动军略在剩余波次持续生效，并可通过自动吞噬、自动进阶与 Set 解锁发展为高阶形态。
- 每次备战至多选择 3 张主动军令进入 HUD；普通/精英波有 2 点军令点，Boss 波有 3 点。
- 军令的上、中、下路目标只读取战前路归属，不随战斗移动变化。
