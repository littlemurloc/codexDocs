# Design Test Record Template

本文件用于记录策划验收、封闭测试和数值调优结果。它不是自动化测试脚本，而是给策划、测试和实装人员在试玩后填写的复盘模板。

使用建议：

- 每次测试至少填写一份“单局记录”。
- 发现明显问题时补一条“问题记录”。
- 做装备、流派或关卡专项测试时，额外填写对应专项表。
- 不追求一次填满所有字段，优先记录能帮助下一轮调整的事实。
