# MVP Design Acceptance Checklist

本清单用于首轮 Y3 原型验收。当前以新版第一版 8 英雄、3x3 站位、英雄小队、升星解锁技能和按英雄关闭自动释放的技能卡牌池为准。旧版技能 6 选 4、旧装备、旧战备 3 选 1 和旧羁绊不作为当前验收项。

## 首轮最低通过口径

- 3 路 3x3 站位、选敌、英雄小队、技能卡牌、波次、备战、奖励和结算状态可以完整跑通一局。
- 8 名首批英雄可进入实机验证。
- 每名英雄 6 技能：3 主动 + 3 被动。
- 玩家可按英雄关闭主动技能自动释放，使该英雄已解锁主动技能进入公共 HUD 卡牌池。
- 默认 2 人位可完成基础难度，4 人位只提供阵容宽度和容错。

## 3x3 站位

- [ ] 我方左侧格位为 `741 / 852 / 963`。
- [ ] 敌方右侧格位为 `147 / 258 / 369`。
- [ ] 前排 = `1/2/3`。
- [ ] 中排 = `4/5/6`。
- [ ] 后排 = `7/8/9`。
- [ ] 侧翼 = `1/4/7` 或 `3/6/9`。
- [ ] 上排优先攻击敌方 `147`。
- [ ] 中排优先攻击敌方 `258`。
- [ ] 下排优先攻击敌方 `369`。
- [ ] 中排无目标时优先支援敌方单位更多的一路，数量相同再随机。
- [ ] 不存在营地生命或推线扣血。

## 英雄小队

- [ ] 每个英雄带领一支小兵队伍。
- [ ] 队长等于英雄。
- [ ] 队长和小兵都是独立可选目标。
- [ ] 玩家方队长死亡后，该队小兵溃败离场。
- [ ] NPC 队长死亡后，该队小兵继续战斗。
- [ ] 小兵死亡不会影响队长。
- [ ] 技能写队长/英雄时，不会误选小兵。
- [ ] 当前小兵兵种至少覆盖近战和远程。

## 技能与卡牌池

- [ ] 每名英雄 6 技能，3 主动 + 3 被动。
- [ ] 技能通过外围升星依次解锁。
- [ ] 英雄进入战斗自动携带所有已解锁技能。
- [ ] 未解锁技能不会在战斗中触发。
- [ ] 被动技能不进入技能卡牌池。
- [ ] 主动技能默认自动释放。
- [ ] 备战阶段可按英雄关闭主动技能自动释放。
- [ ] 关闭某英雄自动释放后，该英雄所有已解锁主动技能进入 HUD 技能卡牌池。
- [ ] 技能卡牌每轮最多发放 4 张。
- [ ] 冷却中、系统锁定或死亡英雄所属技能不会被抽取。
- [ ] 关闭更多英雄自动释放会扩大卡牌池，并降低关键技能进入当前手牌的稳定性。

## 8 英雄机制验收

- [ ] 黄忠：蓄势只在后排稳定获得；3 个主动分别体现单点、穿线、斩首打断。
- [ ] 刘备：所有主动只作用于友方队长，不治疗小兵；仁德阈值和冷却正确。
- [ ] 吕蒙：背刺按目标背后 120 度判断；隐身期间可普攻，主动/受伤/时间结束破隐。
- [ ] 赵云：无独立闪身；成功闪避获得游龙；闪避来源和游龙消耗正确。
- [ ] 张飞：守势提供近战减伤，越被打越能守；消耗守势会失去对应减伤。
- [ ] 夏侯惇：刚烈来自近战承伤，满 4 层自动反斩；技能可推动或消耗刚烈。
- [ ] 吕布：威势来自参与击破队长/精英，小兵不提供威势；斩将消耗威势。
- [ ] 诸葛亮：谋略来自后排守位；区域和一路技能范围不跨全场。

## 旧系统边界

- [ ] 旧版 6 选 4 技能装配不作为当前验收项。
- [ ] 首轮原型不依赖旧版装备通过。
- [ ] 首轮原型不依赖旧版战备 3 选 1 或 30 个战备能力通过。
- [ ] 首轮原型不依赖旧版羁绊通过。
- [ ] 若临时使用旧内容占位，必须标记为 proxy，不作为最终设计。
