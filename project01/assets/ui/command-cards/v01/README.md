# Command Card Art V01

42 张军令卡中央插图资源：12 张主动军令与 30 张被动军略/自动融合结果。所有最终文件为 512x512 RGBA PNG，外缘保留 12% 透明安全区。

- `source/`: 已确认清稿阶段的原始色键图，保留供溯源。
- `transparent/`: 游戏接入用无文字透明主资源。
- `preview/preview_alpha_checker.png`: 透明边缘检查。
- `preview/preview_48px.png`: 常态小尺寸辨识度检查。
- `preview/preview_passive_alpha_checker.png`: 30 张被动军略的透明边缘检查。
- `preview/preview_passive_hud_48.png`: 30 张被动军略的 48px HUD 识别度检查。
- `manifest.json`: 卡牌 ID、文件映射与导出校验。
