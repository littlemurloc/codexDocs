# Market Selection Kit V01

Reusable art for the selected B three-choice command-card screen.

The card base is intentionally text-free. Program content supplies the ability
illustration, Set crest, title, type label, effect summary, price, progress and
localized button copy.

## Directories

- `source/`: no-text chroma-key source art.
- `transparent/`: game-ready RGBA PNG components.
- `preview/`: checkerboard alpha inspections.
- `manifest.json`: component purpose, dimensions and composition order.

## Layer Order

1. Dynamic ability illustration.
2. Dynamic Set crest icon.
3. `choice_card_base`.
4. `choice_card_set_crest_frame`.
5. Dynamic title, Set label, effect summary, price and progress text.
6. Type marker, progress pips and currency coin.
7. Optional Set advance cue.
8. Selected outline and engine bloom.
