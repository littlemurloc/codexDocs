# Market Status Bands v01

Three text-free status-band assets for the lower area of a market card.

- `status_band_discount.png`: red wax seal for dynamic discount text.
- `status_band_set_fusion.png`: jade wax seal for dynamic set-fusion text.
- `status_band_insufficient.png`: charcoal wax seal for dynamic insufficient-funds text.

All deliverables are 512x128 RGBA PNG files. Apply program text inside the clear central parchment area.
