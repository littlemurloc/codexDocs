# Skill Select v02 Icon Crops

These are concept portrait crops extracted from an interim `skill-loadout-v02` UI mockup.

They were generated inside a UI mockup, not as final standalone portraits from `generate-sanguo-portrait-images`. The original interim mockup is not retained in `assets/ui/preview/` because only approved UI images are kept there. Use these crops for layout and prototype validation unless they are later regenerated or approved as formal character assets.

## Folders

- Root files: wider portrait-area crops from the character grid.
- `face-crops/`: tighter face-focused crops for compact icon usage.
