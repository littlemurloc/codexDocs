Use only the PNG files in `png_frames/` for Y3 sequence-frame import.

- Import type: scattered PNG images
- Sort rule: filename ascending
- Frame name pattern: `card_burn_D_000.png` to `card_burn_D_041.png`
- Frame count: 42
- Canvas size: 384 x 640
- Format: optimized indexed PNG with alpha
- Total PNG frame size: 1.36 MB
- Suggested preview playback: 24 fps

The GIF is review-only. Do not import the GIF into Y3.
