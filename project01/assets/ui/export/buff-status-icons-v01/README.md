# 单位 Buff 状态图标 v01

用于单位头顶、状态公告板或特效贴图的 12 个状态图标。

## 规格

- 每张：`256x256` RGBA PNG。
- 图标四边保留透明安全区，可叠加描边、缩放或柔光。
- 无文字、无卡框、无固定底板，适合小尺寸状态展示。
