# 第一批 8 英雄技能图标

- 数量：48 张（8 名英雄，每名 6 技能）
- 规格：`128 x 128` PNG，RGB，不带 UI 外框
- 风格：三国军阵木刻 / 战图插画；宣纸底、墨线、玉绿、土黄和克制的朱砂色
- 技能主源：[第一批 8 个英雄最新版技能](../../../docs/design/first-batch-8-heroes-latest-skills.md)

## 使用

- 最终图标：`icons_128/`
- 文件与技能名称映射：`manifest.json`
- 全量实际尺寸预览：`preview_all_48_128px.png`

图标按小尺寸可读性设计：每张以一个主动作、主姿态或主物件为核心，再用不超过两个机制符号补充效果。人物均为通用动作剪影，不是英雄头像或立绘。
