# 技能插入效果 V03

## 版式硬规则

- 角色或军略主体、方形技能图标、标题条、能量线全部固定在画面左侧 `0-27%` 的信息岛内。
- 中间与右侧至少保留 `73%` 的可操作战场；不得再出现独立的右侧图标。
- 技能图标使用现有 `1:1` 方形资源，置于主体下缘或下方，与主体形成同一组信息。
- 标题条仅用于名称和短标签，宽度不越过信息岛。
- 武将插入使用半身立绘加单一方向笔触；军略插入统一使用竖向竹简与铜印，不使用人物。

## 军略统一色调

| 类别 | 主色 | 辅色 | 典型表现 |
| --- | --- | --- | --- |
| 攻城、火攻、爆发 | 朱砂红 | 焦金、墨黑 | 火痕、飞灰、冲击笔触 |
| 阵法、谋略、控制 | 靛紫 | 月白、墨黑 | 阵纹、符纸、回旋墨气 |
| 治疗、增益、恢复 | 青绿 | 玉白、淡金 | 流云、草木、柔和扩散 |
| 箭雨、号令、行军 | 深青蓝 | 银白、冷金 | 风痕、旗影、锐利线势 |
| 防御、据守、地形 | 土褐 | 黯金、岩灰 | 山石、城垣、厚重皴擦 |

所有军略共享同一支竹简、铜印、标题条与方形图标框；仅更改笔触色、图标内容及少量环境纹理。
